About me.

I'm a junior web developer with an interest in responsive design and best practices in web development. I have a strong passion for my work, and I love the constant growth and opportunities for learning that it provides me.

I entered the IT world initially as a Recruiter, but found myself incredibly drawn to technical side of things. I decided to enroll in a full-stack Java/SQL development course with the Arkansas Coding Academy. During my path of learning how things work from front to back, in everything from writing code in the models to get data from the database all the way through presenting that data with HTML and CSS in the views, I have gained a valuable knowledge of what it takes to get the job done in the front end and in the back end.

Since graduating from the Arkansas Coding Academy, I have focused deeper into learning best practices on the front end. I love being able to manipulate elements visually to create a beautiful, user-friendly design, and responsive interface that works across different browsers/devices.
